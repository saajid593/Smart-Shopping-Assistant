# 🛍️ Smart Shopping Assistant (SmartCart)

AI-powered web app to:
- Search products
- Compare prices
- Track expenses
- Get AI budget suggestions

## Tech Stack
React.js, Node.js, MongoDB, Tailwind CSS, Chart.js, Claude AI

## Features
- Real-time search
- Price comparison
- Expense dashboard
- AI recommendations
